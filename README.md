# SymbiocreationUi

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.6.

## Prerequisites

- Node.js 18.x or 20.x (recommended: 20.15.0)
- npm 10.x

## Installation

```bash
# Install dependencies (use legacy-peer-deps due to @angular/flex-layout compatibility)
npm install --legacy-peer-deps

# Install compatible TypeScript version
npm install typescript@5.4.5 --save-dev --legacy-peer-deps
```

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

```bash
ng serve -o
```

## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory.

ng build

# Production build (uses environment.prod.ts)
ng build --configuration=production
